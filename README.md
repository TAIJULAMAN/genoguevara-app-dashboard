# Step 11 Admin Dashboard

A premium, modern admin dashboard built with **React**, **Vite**, and **Tailwind CSS**. This dashboard is designed for managing spiritual wellness content, featuring a sleek user interface with glassmorphism effects, dynamic components, and a focus on visual excellence.
